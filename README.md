# Personal Portfolio Website

A modern, responsive portfolio website built to showcase my projects, skills, experience, and professional background.

## Live Demo

🌐 Website: https://michaelalt2304.github.io/

## Features

- Responsive design for desktop, tablet, and mobile devices
- Project showcase section
- Skills and technologies overview
- Contact information and social links
- Clean and modern UI
- Fast-loading static website

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages


## Deployment

This project is deployed using GitHub Pages.

To deploy updates:

1. Push changes to the `main` branch.
2. GitHub Pages will automatically rebuild and publish the site.

## Customization

You can customize:

- Personal information
- Project descriptions
- Social media links
- Resume/CV links
- Color themes and styling
- Images and assets
